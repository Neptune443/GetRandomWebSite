PSForms

Why learn another product?

With PSForms from Get-Random you do not need to learn another product. Traditionally you install some product in which you build your forms, and it comes with an SDK in which you call out to your logic which automates the task. 

With PSForms you do not need any of that. Build your code in PowerShell and we generate the form for you. That is it. We analyse the param() block of your code and build the form for you. 

Well, we said you did not have to learn a new product, this of course never entirely true. We do have features to manage proxy or "run as" accounts. We do have additional configuration in order to control the web UI elements, but the key thing is. All this is done in PowerShell. 

It is easy, so easy that sometimes you can take existing scripts and just add them as functions to a module and the whole script has been published as a self service form.
